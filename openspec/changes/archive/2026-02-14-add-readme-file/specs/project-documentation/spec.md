## ADDED Requirements

### Requirement: README Presence

The project MUST have a `README.md` file in the root directory.

#### Scenario: Project Initialization

- **WHEN** a user clones or downloads the project repository
- **THEN** a `README.md` file must be present in the project's root directory.

### Requirement: README Content

The `README.md` file MUST contain essential project information.

#### Scenario: Reading the README

- **WHEN** a user opens the `README.md` file
- **THEN** they should see a project title and a brief description.
- **AND** they should find instructions on how to build or run the project.
