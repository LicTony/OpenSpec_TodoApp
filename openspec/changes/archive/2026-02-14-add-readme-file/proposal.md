## Why

El proyecto actual carece de documentación básica, lo que dificulta la comprensión de su propósito y cómo comenzar a usarlo para nuevos colaboradores o incluso para el propio desarrollador en el futuro. Añadir un `README.md` es un paso fundamental para mejorar la capacidad de mantenimiento y la colaboración.

## What Changes

- A new `README.md` file SHALL be created in the project root.
- The file SHALL contain a concise project description, its purpose, and basic setup or usage instructions.
